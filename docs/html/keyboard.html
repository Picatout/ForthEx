?PRTCHAR ( n -- f )

Vérifie si 'n' est une caractère imprimable dans l'intervalle {32..126}
et retourne un indicateur booléen.

arguments:
n    Entier simple
retourne:
f   Indicateur booléen, vrai si n → {32..126}
